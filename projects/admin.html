필터/검색 기능

검색 기능

EXPLAINATION

관리자페이지에서 관리자가 페이지 운영에 관리가 편하게 검색이 필요한 관리페이지에 검색기능을 추가하였습니다. 상품 목록 / 회원목록 / 상품후기목록 / 게시판 목록 4개의 곳에 검색 기능을 추가하였고, 동영상 시연은 밑 필터 기능을 시연하는 동영상에 같이 첨부하였습니다.

필터 기능

EXPLAINATION

위 동영상은 상품관리페이지에서 상품 카테고리별 검색과 상품명 검색을 보여주는 동영상입니다.

페이지네이션

EXPLAINATION

관리자페이지에서 한페이지당 게시글 5개씩으로 배치 지정을하여 게시글(상품, 회원 등)의 갯수에 따라 페이지네이션이 늘이나는 Controller를 작성하였습니다.

상품등록/삭제/수정

상품등록/삭제

EXPLAINATION

관리자페이지 상품등록을 하는 동영상입니다. 입력창에 맞게 상품을 등록한 후 상품관리페이지로 들어가면 상품이 등록되어있는 것을 확인할 수 있습니다. 또한 "상품삭제" 버튼을 클릭하면 상품이 삭제가 되고 검색을 해보면 삭제가 되었으니 검색이 되지가 않는 것을 확인할 수 있습니다.

상품수정

EXPLAINATION

관리자페이지 상품 수정 테스트 동영상입니다. 입력창에 맞게 상품을 수정 내용을 입력 후 수정버튼을 클릭시 상품이 수정된 것을 확인할 수 있습니다.
